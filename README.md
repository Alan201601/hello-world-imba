# hello-world-imba

Tiny template for testing out Imba in the browser.

## Getting started

### Install Imba
```bash
npm install -g imba
```

### Clone repository
```bash
git clone https://github.com/somebee/hello-world-imba.git
```

### Enter repository
```bash
cd hello-world-imba
```

### Install dependencies
```bash
npm install
```

### Watch to recompile
```bash
npm run watch
```

## Run
Open `index.html` in your browser (you should not need a webserver).
